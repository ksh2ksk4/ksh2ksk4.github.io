Raspberry Pi 4 (Memory: 2GB)に openmediavault をインストールして構築する。

2 Raspbian

Raspbian Buster Lite (Version: September 2019) をインストールしてヘッドレスで利用する。

  • imgファイルをダウンロード
  • balenaEtcher でマイクロSDカードへ書き込む

2.0.2 Settings

2.0.2.1 raspi-config
$ sudo raspi-config nonint do_expand_rootfs
$ sudo raspi-config nonint do_hostname apricot
$ sudo raspi-config nonint do_change_timezone Asia/Tokyo
$ sudo raspi-config nonint do_change_locale ja_JP.UTF-8
$ sudo raspi-config nonint do_wifi_country Japan
$ sudo raspi-config nonint do_memory_split 16
$ sudo reboot
2.0.2.2 apt-get
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo reboot
2.0.2.3 DHCP

ネットワーク設定はDHCPを利用するが、スタティックIPアドレスになるようルータ(NVR510)を設定する。

dhcp scope bind 1 192.168.126.10 ethernet dc:a6:32:xx:xx:xx
ip host apricot.home 192.168.126.10

dc:a6:32 は Raspberry Pi Trading Ltd のベンダID。

2.0.2.4 SSH
2.0.2.4.1 ssh グループへの追加

OMVをインストールするとユーザ pi でログインできなくなる。
(インストール後はOMVがSSHを管理するようになり、 ssh グループに属していないユーザのログインを許可していない模様)

OMVインストール後もユーザ pi でログインできるよう、予め以下を実行しておく。

$ sudo usermod -a -G ssh pi

これを実行せずOMVをインストールした場合、OMVの管理ページでユーザ pi を ssh グループに追加する。

2.0.2.4.2 パスワードレスでログイン

作業用PCで以下を実行する。

$ vi ~/.ssh/config

以下を追記。

Host apricot.home
  HostName apricot.home
  User pi
  IdentityFile ~/.ssh/keys/apricot.home.key

$ ssh-keygen -t ecdsa -b 521 -f ./apricot.home.key -C 'apricot.home'
$ ssh-copy-id -i ./apricot.home.key pi@apricot.home

これでパスワードレスにログインできるはず。

$ ssh apricot.home
2.0.2.4.3 パスワードログインを不許可
$ sudo vi /etc/ssh/ssh_config
$ sudo systemctl restart ssh

ssh_config の変更内容はBookmarksを参照。

2.0.2.5 ログインメッセージ

これも鬱陶しいので消しておく。

$ sudo vi /etc/motd
2.0.2.6 Swap

スワップを無効化。

$ free -h
$ sudo swapoff --all
$ free -h

$ sudo systemctl status dphys-swapfile
$ sudo systemctl stop dphys-swapfile
$ sudo systemctl disable dphys-swapfile
$ sudo systemctl status dphys-swapfile
2.0.2.7 Wifi

利用しないので無効化。

$ sudo vi /boot/config.txt

以下を追記。

dtoverlay=disable-wifi

$ sudo reboot
2.0.2.8 Buletooth

利用しないので無効化。

$ sudo vi /boot/config.txt

以下を追記。

dtoverlay=disable-bt

$ sudo reboot
2.0.2.9 Avahi
$ sudo systemctl stop avahi-daemon
$ sudo systemctl disable avahi-daemon
$ sudo systemctl status avahi-daemon
$ sudo reboot

2.0.3 Backup

マイクロSDカードを作業用PCへ接続して以下を実行する。
(デバイスは環境によって異なる)

$ sudo dd if=/dev/disk2 of=./apricot.img bs=16M

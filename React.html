3.2 第2章

3.2.1 どちらも同じ

ReactDOM.render(<TodoApp />, document.querySelector("#app"))

ReactDOM.render(React.createElement(TodoApp), document.querySelector('#app'))

3.2.2 開始タグと終了タグの間にある子ノードは this.props.children で参照できる

class Foo extends React.Component {
  render() {
    return (
      <div>
        <h1>{this.props.children}</h1>
      </div>
    )
  }
}

ReactDOM.render(<Foo>foo</Foo>, document.querySelector("#app"))

3.2.3 DOMに存在しない属性

  • key
    • 第11章参照
  • ref
    • 第6章参照

  • dangerouslySetInnerHTML 

    class Foo extends React.Component {
  render() {
    let htmlString = {
      __html: '<span>HTML string</span>'
    }
    return (
      <div dangerouslySetInnerHTML={htmlString}></div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

3.2.4 イベント

class Foo extends React.Component {
  handleClick() {
    alert('foo')
  }

  render() {
    return (
      <div onClick={this.handleClick}>Click me!</div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

3.2.5 コメント

class Foo extends React.Component {
  render() {
    return (
      <div
        /**
         * インラインコメント
         *
         * 複数行のコメントも書ける
         */
        name="foo" // 一行のインラインコメント
        >
        {/**
          * 子ノードとしてのコメント
          *
          * 複数行のコメントも書ける
          */}
        foo
      </div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

3.3 第3章

3.3.1 ライフサイクルメソッド

コンポーネントにおいて render メソッドは必ず実装しなければならない。

3.3.1.1 最初のコンポーネントインスタンス生成時
  • getDefaultProps
    • このメソッドが実行されるのは最初だけ
  • getInitialState
  • componentWillMount
  • render
  • componentDidMount
3.3.1.2 以降のコンポーネントインスタンス生成時
  • getInitialState
  • componentWillMount
  • render
  • componentDidMount
3.3.1.3 コンポーネントインスタンス更新時
  • componentWillReceiveProps
  • shouldComponentUpdate
  • componentWillUpdate
  • render
  • componentDidUpdate
3.3.1.4 コンポーネントインスタンス破棄時
  • componentWillUnmount

3.4 第4章

3.4.1 props

  • コンポーネントの内部で変更してはいけない
3.4.1.1 スプレッド演算子
class Foo extends React.Component {
  render() {
    let props = {
      id: 'foo',
      name: 'bar'
    }

    return (
      <div {...props} />
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))
3.4.1.2 PropTypes

React v15.5から別のパッケージになっている。

3.4.1.3 getDefaultProps

プロパティのデフォルト値を定義するメソッド。
デフォルト値はすべてのコンポーネントインスタンスで共通。

3.4.2 state

  • コンポーネントの内部でのみ使用される
  • getInitialState メソッドで初期化する
  • setState メソッドで変更する
    • state を直接変更してはいけない
    • このメソッドの後で render メソッドが必ず実行される
      • render メソッドの実行で仮想DOMが更新され、最終的に実際のDOMが更新される

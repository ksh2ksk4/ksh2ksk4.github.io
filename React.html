1.2 第2章

1.2.1 どちらも同じ

ReactDOM.render(<TodoApp />, document.querySelector("#app"))

ReactDOM.render(React.createElement(TodoApp), document.querySelector('#app'))

1.2.2 開始タグと終了タグの間にある子ノードは this.props.children で参照できる

class Foo extends React.Component {
  render() {
    return (
      <div>
        <h1>{this.props.children}</h1>
      </div>
    )
  }
}

ReactDOM.render(<Foo>foo</Foo>, document.querySelector("#app"))

1.2.3 DOMに存在しない属性

  • key
    • 第11章参照
  • ref
    • 第6章参照

  • dangerouslySetInnerHTML 

    class Foo extends React.Component {
  render() {
    let htmlString = {
      __html: '<span>HTML string</span>'
    }
    return (
      <div dangerouslySetInnerHTML={htmlString}></div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

1.2.4 イベント

class Foo extends React.Component {
  handleClick() {
    alert('foo')
  }

  render() {
    return (
      <div onClick={this.handleClick}>Click me!</div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

1.2.5 コメント

class Foo extends React.Component {
  render() {
    return (
      <div
        /**
         * インラインコメント
         *
         * 複数行のコメントも書ける
         */
        name="foo" // 一行のインラインコメント
        >
        {/**
          * 子ノードとしてのコメント
          *
          * 複数行のコメントも書ける
          */}
        foo
      </div>
    )
  }
}

ReactDOM.render(<Foo />, document.querySelector("#app"))

1.2.6 特別な属性

HTMLにおける for, class はJSXでは htmlFor, className になる。

1.3.1 ライフサイクルメソッド

コンポーネントにおいて render メソッドは必ず実装しなければならない。

1.3.1.1 最初のコンポーネントインスタンス生成時

  • componentWillMount
  • render
  • componentDidMount
1.3.1.2 以降のコンポーネントインスタンス生成時
  • getInitialState
  • componentWillMount
  • render
  • componentDidMount
1.3.1.3 コンポーネントインスタンス更新時
  • componentWillReceiveProps
  • shouldComponentUpdate
  • componentWillUpdate
  • render
  • componentDidUpdate
1.3.1.4 コンポーネントインスタンス破棄時

1 Install

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

/usr/local/sbin にパスを通す1。

$ set -U fish_user_paths /usr/local/sbin $fish_user_paths
$ set -S fish_user_paths

HomebrewのAnalyticsをオフ。

$ brew analytics state
$ brew analytics off
$ brew analytics state

2 Packages

2.1 fish 2

2.2 Coreutils

$ brew install coreutils

2.2.1 注意点

  • g プレフィックスのコマンドを使う場合
    • /usr/local/bin にパスを通す
      デフォルトでパスは通っているはず。
  • g プレフィックスのないコマンドを使う場合

    • /usr/local/opt/coreutils/libexec/gnubin にパスを通す

      $ set -U fish_user_paths (brew --prefix coreutils)/libexec/gnubin $fish_user_paths

2.3 Spacemacs 2

依存関係で SQLite がインストールされるため、以下を実行しておく。

$ set -U fish_user_paths (brew --prefix sqlite)/bin $fish_user_paths

2.4 less

$ brew install less lesspipe

set -x PAGER less
set -x LESS -ij5gFMRSWX
set -x LESSOPEN '|/usr/local/bin/lesspipe.sh %s'
set -x LESS_ADVANCED_PREPROCESSOR 1

$ mv ~/.tmux.conf ~/.config/tmux/
$ ln -sfv ~/.config/tmux/.tmux.conf

3 Commands

3.1 Show dependencies for specified formulae

$ brew deps --tree formulae

3.2 Show the formulae that uses specified formulae

$ brew uses --installed formulae

3.3 Show the dependencies between installed formulae

$ for i in $(brew list); do echo "[${i}]"; brew uses --installed ${i}; echo ""; done

3.4 Uninstall

$ brew uninstall formulae $(brew uses --installed formulae)

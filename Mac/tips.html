1 不可視ファイル

1.1 可視化

$ defaults write com.apple.finder AppleShowAllFiles -bool TRUE
$ killall Finder

1.2 不可視化

$ defaults delete com.apple.finder AppleShowAllFile
$ killall Finder

2 .DS_Storeファイル

2.1 ネットワークドライブに作成しない

$ defaults write com.apple.desktopservices DSDontWriteNetworkStores -bool TRUE
$ killall Finder

元に戻すには下記の通り。

$ defaults write com.apple.desktopservices DSDontWriteNetworkStores -bool FALSE
$ killall Finder

2.2 USBドライブに作成しない

$ defaults write com.apple.desktopservices DSDontWriteUSBStores -bool TRUE
$ killall Finder

元に戻すには下記の通り。

$ defaults write com.apple.desktopservices DSDontWriteUSBStores -bool FALSE
$ killall Finder

3 不要なディレクトリ1を削除

$ sudo rm -rf Documents Movies Music Public

4 Finder

  • ファイルのコピー
    • command + V
  • ファイルの移動
    • option + command + V

    • 対象ファイルを選択して option + command + C
  • 不可視ファイルの表示・非表示を切り替え
    • command + shift + .

5 Time Machine

5.1 バックアップに時間が掛かる場合

  • セキュリティソフトのリアルタイムファイルシステム保護を一時的に無効にする
    • この間はネットワーク接続も切断しておく

  • バックアップファイルのインデックスを作成しないようにする
    状態を確認。

    $ mdutil -sa

    インデックスを作成しないように設定を変更。

    $ sudo mdutil -i off /Volumes/SSD/Backups.backupdb

    状態を再確認。

    $ mdutil -sa

    多分、設定は変わっていないはず。
    ただ、 mdworker プロセスが動かなくなるため処理は早くなる。

  • バックアップの優先順位を一時的に上げる

    $ sudo sysctl debug.lowpri_throttle_enabled=0

    バックアップが完了したら元に戻す。

    $ sudo sysctl debug.lowpri_throttle_enabled=1

6 スクリーンショットに影を付けない

$ defaults write com.apple.screencapture disable-shadow -bool true

7 OSアップデートの通知を非表示にする

Catalinaのアップデート通知を非表示にする。

$ sudo /usr/sbin/softwareupdate --ignore "macOS Catalina"
$ defaults write com.apple.systempreferences AttentionPrefBundleIDs -dict com.apple.preferences.softwareupdate 0
$ killall Dock

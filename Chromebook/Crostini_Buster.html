2.2.2.3.4 dotspacemacs/user-config







2.2.2.3.4.1 日本語入力




Google 日本語入力 を利用する。






emacs-mozc-bin をインストール。





$ sudo apt install emacs-mozc-bin





dotspacemacs-additional-packages に mozc と mozc-temp を追加。







dotspacemacs-additional-packages
'(
  mozc
  mozc-temp
  )







dotspacemacs/user-config に以下を追加。







(set-language-environment "Japanese")
(prefer-coding-system 'utf-8-unix)

(use-package mozc
  :init
  (setq default-input-method "japanese-mozc")
  (setq mozc-helper-program-name (cond ((eq system-type 'darwin) "~/.own/bin/mozc_emacs_helper")
                                       ((eq system-type 'gnu/linux) "/usr/bin/mozc_emacs_helper")))
  :custom
  (mozc-candidate-style 'echo-area)
  (mozc-leim-title "[も]"))

(use-package mozc-temp
  :init
  (global-set-key (kbd "M-n") 'mozc-temp-convert)
  :custom
  (mozc-temp-remove-pre-space nil))











2.2.2.3.4.2 Unicode正規化 のNFD<->NFC対応




主にMacとデータをやりとりする場合のケア。







(require 'ucs-normalize)
(when (eq system-type 'darwin)
  (set-file-name-coding-system 'utf-8-hfs)
  (setq locale-coding-system 'utf-8-hfs))
(when (eq system-type 'gnu/linux)
  (set-file-name-coding-system 'utf-8-unix)
  (setq locale-coding-system 'utf-8-unix))

2.2.5.1 インストール




$ cd ~/Downloads
$ curl -LO 'https://repo1.maven.org/maven2/org/eclipse/jetty/jetty-distribution/9.4.30.v20200611/jetty-distribution-9.4.30.v20200611.zip'
$ unzip jetty-distribution-9.4.30.v20200611.zip

$ cd
$ mv Downloads/jetty-distribution-9.4.30.v20200611 ./
$ ln -sfv jetty-distribution-9.4.30.v20200611 jetty










2.2.5.2 設定






set -x JETTY_HOME ~/jetty
set -x JETTY_BASE {$JETTY_HOME}/doc-base












2.2.5.3 スタートアップスクリプト






function startup -d 'My startups'
  cd $JETTY_BASE
  java -jar {$JETTY_HOME}/start.jar --module=http,deploy >/dev/null 2>&1 &

  return 0
end












2.2.5.4 使い方





スタートアップスクリプトを実行してJettyを起動。





$ startup

$ ip addr

2.2.9.2 ロケール





ビルドとかすると以下のようなロケールのエラーが発生する。





perl: warning: Setting locale failed.
perl: warning: Please check that your locale settings:
        LANGUAGE = (unset),
        LC_ALL = (unset),
        LANG = "ja_JP.UTF-8"
    are supported and installed on your system.
perl: warning: Falling back to the standard locale ("C").
locale: Cannot set LC_CTYPE to default locale: No such file or directory
locale: Cannot set LC_MESSAGES to default locale: No such file or directory
locale: Cannot set LC_ALL to default locale: No such file or directory





鬱陶しいのでロケールを設定する。





$ sudo dpkg-reconfigure locales





ja_JP.UTF-8 が選択されていないので、 ja_JP.UTF-8 を選択する。

(システムデフォルトのロケールは en_US.UTF-8 にしておく)






これでエラーは発生しなくなるはず。

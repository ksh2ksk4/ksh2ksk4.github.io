2 fish

2.1 Install

$ sudo apt install fish

2.2 Change the login shell

$ less /etc/shells
$ chsh -s /usr/bin/fish
Password:
chsh: PAM: Authentication failure

ログインシェルを変更しようとするとこうなる。
(パスワード未設定だからっぽい)

こうする。

$ sudo su -
# chsh -s /usr/bin/fish ksh2ksk4

ログインシェル変更後はChromebookを再起動する。
(再起動しないと挙動がおかしい)

$ vi ~/.config/fish/functions/fish_user_key_bindings.fish

function fish_user_key_bindings -d 'Define own key bindings'
  bind \cr 'peco_select_history (commandline -b)'
end

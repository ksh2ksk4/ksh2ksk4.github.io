2 特徴的な概念・機能

2.1 Shadowing

前に定義した変数と同じ名前の新しい変数を定義できる。 

let x = 1;
let x = x + 1;

ミュータブルではないので再束縛は不可(コンパイルエラー)。 

x = 100;

型を変更することも可能。 

let spaces = "        ";
let spaces = spaces.len();

2.2 Array

固定長。 可変長のデータを扱う場合はベクタを利用する。

2.3 Statement/Expression

  • 文
    • 実行後に値を返さない
    • 終端に ; を含む
  • 式
    • 実行後に値を返す
    • 終端に ; を含まない

2 Data types

2.1 Scalar types

2.1.1 整数型(符号あり)

  Size(bit) Min Max
i8 8 -256 127
i16 16 -32768 32767
i32 32 -2147483648 2147483647
i64 64 -9223372036854775808 9223372036854775807
isize 1      

取り得る値の範囲は以下の通り。

\[ -(2^{n-1}) \leq x \leq 2^{n-1} - 1 \]

i32 が整数型の基準型。

2.1.2 整数型(符号なし)

  Size(bit) Min Max
u8 8 0 255
u16 16 0 65535
u32 32 0 4294967295
u64 64 0 18446744073709551615
usize 1      

取り得る値の範囲は以下の通り。

\[ 0 \leq x \leq 2^n - 1 \]

2.1.3 浮動小数点数型

f32 (単精度浮動小数点数) と f64 (倍精度浮動小数点数) があり、 f64 が浮動小数点数型の基準型。

2.1.4 論理値型

論理値型は bool であり、 true と false が取り得る値。

2.1.5 文字型

文字型は char であり、Unicodeのスカラ値を表す。
文字列とは異なるため、 " ではなく ' でクォートする。

2.1.6 ユニット型

ユニット型は () であり、空のタプル( () )が取り得る値。

2.2.1 タプル型

2.2.2 配列型

  • コレクションの一種
  • 全要素は同一の型
  • 固定長

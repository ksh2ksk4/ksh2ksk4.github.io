2 言語仕様

2.1 Data types

2.1.1 Scalar types

  1. 整数型(符号あり)
      Size(bit) Min Max
    i8 8 -256 127
    i16 16 -32768 32767
    i32 32 -2147483648 2147483647
    i64 64 -9223372036854775808 9223372036854775807
    isize 1      

    取り得る値の範囲は以下の通り。

    \[ -(2^{n-1}) \leq x \leq 2^{n-1} - 1 \]

    i32 が整数型の基準型。

  2. 整数型(符号なし)
      Size(bit) Min Max
    u8 8 0 255
    u16 16 0 65535
    u32 32 0 4294967295
    u64 64 0 18446744073709551615
    usize 1      

    取り得る値の範囲は以下の通り。

    \[ 0 \leq x \leq 2^n - 1 \]

  3. 浮動小数点数型

    f32 (単精度浮動小数点数) と f64 (倍精度浮動小数点数) があり、 f64 が浮動小数点数型の基準型。

  4. 論理値型

    論理値型は bool であり、 true と false が取り得る値。

  5. 文字型

    文字型は char であり、Unicodeのスカラ値を表す。
    文字列とは異なるため、 " ではなく ' でクォートする。

  6. ユニット型

    ユニット型は () であり、空のタプル( () )が取り得る値。

2.2 Variable

2.2.1 Shadowing

前に定義した変数と同じ名前の新しい変数を定義できる。 

let x = 1;
println!("The value of x is {}", x);
let x = x + 1;
println!("The value of x is {}", x);

The value of x is 1
The value of x is 2

ミュータブルではないので再束縛は不可(コンパイルエラー)。 

let x = 1;
x = 100;

error[E0384]: cannot assign twice to immutable variable `x`
 --> rust-src-KLc4WQ.rs:3:1
  |
2 | let x = 1;
  |     - first assignment to `x`
3 | x = 100;
  | ^^^^^^^ cannot assign twice to immutable variable

型を変更することも可能。 

let spaces = "        ";
let spaces = spaces.len();
println!("The value of spaces is {}", spaces);

The value of spaces is 8

2.3 Function

  • main はプログラムのエントリポイント
  • snake_case で命名するのが慣例
  • 関数定義では仮引数2の型は宣言しなければならない
  • 戻り値は -> の後に型を指定して宣言しなければならない
fn main() {
    println!("The return value of sub_function is {}", sub_function(10));
}

fn sub_function(x: i32) -> i32 {
    println!("The value of x is {}", x);
    x
}

The value of x is 10
The return value of sub_function is 10
  • 文(statement)
    • 実行後に値を返さない
    • 終端に ; を含む
    • 関数定義は文
  • 式(expression)
    • 実行後に値を返す
    • 終端に ; を含まない
    • ブロック( {} )は式

2.4 フロー制御

2.4.1 if

条件式は bool でなければならない。
(bool 以外の値が暗黙的に変換されることはない) 

if true {
    println!("true");
} else {
    println!("false");
}

true

条件が複数存在する場合、 else if を利用できる。
また、 if は文ではなく式であるため、文や式の右辺に置くことができる。 

let result = if true {
    true
} else {
    false
};

println!("The value of result is {}", result);

The value of result is true

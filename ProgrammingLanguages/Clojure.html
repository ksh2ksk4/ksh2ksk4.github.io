1 Clojureの日本語ガイド

1.1 作業ログ

プロジェクトディレクトリを作成。

$ cd MyGoogleDrive/Clojure
$ mkdir clj_todo
$ cd clj_todo
$ mkdir -p resource src/{clj,cljs}/todo

Bootの設定。

$ boot -h
$ boot -V
$ boot -V >boot.properties
$ vim boot.properties

BOOT_CLOJURE_VERSION=1.8.0 -> BOOT_CLOJURE_VERSION=1.9.0

$ vim build.boot

(set-env!
 :source-paths #{"src/clj" "src/cljs"}
 :resource-paths #{"resource"}
 :dependencies '[[org.clojure/clojure "1.9.0"]
                 [org.clojure/clojurescript "1.9.946"]])

(task-options!
 target {:dir #{"target"}})

(deftask dev
  "Build clj-todo project."
  []
  (comp (jar) (target)))

$ vim src/clj/todo/core.clj

(ns todo.core)

$ boot dev

あとは進めていく。

1.2 注意事項

  • proneはBoot環境でも動くが、プロジェクトルートディレクトリにproject.cljファイルがないとエラーになる
    • 空のファイルでOK

2 Spacemacs

2.1 CIDER

  • cider-jack-in
    • M-RET '
    • nREPLへ接続する
  • cider-jack-in-clojurescript
    • M-RET "
    • nREPLへ接続する
  • cider-eval-last-sexp
    • C-x C-e
    • S式の評価結果をインライン表示する
  • cider-eval-print-last-sexp
    • S式の評価結果を表示する(ソースファイルへ書き込む)
  • cider-eval-last-sexp-to-repl
    • C-c M-e
    • S式の評価結果をnREPLへ送る
  • cider-insert-last-sexp-in-repl
    • C-c M-p
    • S式をnREPLへ送る
      • 評価はしない
  • cider-pprint-eval-last-sexp-to-repl
    • S式の評価結果をpretty-printedでnREPLに表示する
    • pprint -> pretty-printed
      • どういうことかは不明
  • cider-pprint-eval-last-sexp
    • C-c C-p
    • S式の評価結果をpretty-printedで表示する
      • 個別バッファ(cider-result)に表示された

1 Getting Started Tutorial

1.1 拡張機能のインストール

1.1.1 マニフェストファイルを作成

{
  "manifest_version": 2,
  "name": "Getting Started Example",
  "version": "1.0",

  "description": "Build an Extension!"
}

1.1.2 Chromeにインストール

  • chrome://extensions をオープン
  • デベロッパー モード を有効化
  • パッケージ化されていない拡張機能を読み込む でインストール

ツールバーにアイコン(デフォルト)が表示される。

1.2 機能追加

1.2.1 バックグラウンドスクリプトを作成

  • ストレージに color を設定する
  • コンソールログを出力する
'use strict';

chrome.runtime.onInstalled.addListener(function() {
  chrome.storage.sync.set({color: '#3aa757'}, function() {
    console.log('The color is #3aa757.');
  });
});

1.2.2 マニフェストファイルを更新

  • storage APIの利用を許可する
  • バックグラウンドスクリプトを登録する
{
  "manifest_version": 2,
  "name": "Getting Started Example",
  "version": "1.0",

  "description": "Build an Extension!",

  "background": {
    "scripts": ["background.js"],
    "persistent": false
  },
  "permissions": ["storage"]
}

1.2.3 拡張機能をリロード

拡張機能の バックグラウンド ページ をクリックするとバックグラウンドスクリプトの挙動を確認できる。

1.3.4 バックグラウンドスクリプトにdeclarativeContent APIのスクリプトを追加

この内容がよくわからない。

'use strict';

chrome.runtime.onInstalled.addListener(function() {
  chrome.storage.sync.set({color: '#3aa757'}, function() {
    console.log('The color is #3aa757.');
  });
  chrome.declarativeContent.onPageChanged.removeRules(undefined, function() {
    chrome.declarativeContent.onPageChanged.addRules([{
      conditions: [new chrome.declarativeContent.PageStateMatcher({
        pageUrl: {hostEquals: 'developer.chrome.com'},
      })],
      actions: [new chrome.declarativeContent.ShowPageAction()]
    }]);
  });
});

1.4 機能追加

1.4.1 ポップアップのボタンをクリックしたときの処理を実装

let changeColor = document.getElementById('changeColor');

chrome.storage.sync.get('color', function(data) {
  changeColor.style.backgroundColor = data.color;
  changeColor.setAttribute('value', data.color);
});

changeColor.onclick = function(element) {
  let color = element.target.value;

  chrome.tabs.query({active: true,
                     currentWindow: true}, function(tabs) {
                       chrome.tabs.executeScript(
                         tabs[0].id,
                         {code: 'document.body.style.backgroundColor = "' + color + '";'});
                     });
};

1.5 ユーザオプション

1.5.1 オプションページを作成

<!DOCTYPE html>
<html>
  <head>
    <style>
      button {
        height: 30px;
        width: 30px;
        outline: none;
        margin: 10px;
      }
    </style>
  </head>
  <body>
    <div id="buttonDiv"></div>
    <div>
      <p>Choose a different background color</p>
    </div>
  </body>
  <script src="options.js"></script>
</html>

1.5.2 オプションの処理を実装

let page = document.getElementById('buttonDiv');
const kButtonColors = ['#3aa757', '#e8453c', '#f9bb2d', '#4688f1'];

function constructOptions(kButtonColors) {
  for (let item of kButtonColors) {
    let button = document.createElement('button');
    button.style.backgroundColor = item;
    button.addEventListener('click', function() {
      chrome.storage.sync.set({color: item}, function() {
        console.log('color is ' + item);
      });
    });
    page.appendChild(button);
  }
}

constructOptions(kButtonColors);

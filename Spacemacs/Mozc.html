1 MacOSの場合

MacOSでSpacemacsを使用する場合、 Google 日本語入力 をインストールすれば日本語を入力できるが、これだとMacOSのIMを経由してGoogle 日本語入力を使うことになる。
この場合、日本語入力状態で C-w とかすると ｗ (全角文字の w)がバッファに入力されてしまいとても煩わしい。

SpacemacsのIM(LEIM)からGoogle 日本語入力を使うとこの現象を回避できるうえ、Spacemacsの日本語入力用パッケージも使えるようになる。
LEIM経由でGoogle 日本語入力を使えるようにするためmozc_emacs_helperを導入する。

1.1 mozc_emacs_helper

1.1.1 事前準備

ビルドにはXcodeとNinjaが必要。
XcodeはApp Storeからインストール。

Ninjaは以下の通り。

$ brew install ninja

1.2 Spacemacs

Spacemacsには以下を設定する。

;;
;; Mozc
;; https://github.com/google/mozc
;;
;; MacOSでIMの日本語入力状態を解除するための設定
(when (eq system-type 'darwin)
  (defun my/eisuu-key ()
    "英数キーをエミュレートする"
    (interactive)
    (call-process "osascript" nil t nil "-e" "tell application \"System Events\" to key code 102"))
  ;; Spacemacsにフォーカスを当てたら英数キーを押す
  (add-hook 'focus-in-hook 'my/eisuu-key)
  ;; Mozcを起動したら英数キーを押す
  (add-hook 'mozc-mode-hook 'my/eisuu-key))

(use-package mozc
  :init
  (setq default-input-method "japanese-mozc")
  (setq mozc-helper-program-name (cond ((eq system-type 'darwin) "~/.own/bin/mozc_emacs_helper")
                                       ((eq system-type 'gnu/linux) "/usr/bin/mozc_emacs_helper")))
  :custom
  (mozc-candidate-style 'echo-area)
  (mozc-leim-title "[も]"))

(use-package mozc-temp
  :init
  (global-set-key (kbd "M-n") 'mozc-temp-convert))

1.3 Karabiner-Elements

ここまでの設定によりSpacemacsでは C-\ で日本語入力できるようになるが、MacOSのIMのキーバインディングを C-\ にしてしまうとLEIMは使えなくなってしまう。
(C-\ のキー入力をMacOSに取られてしまいSpacemacsには渡らなくなってしまうため)

ただ、Spacemacsとそれ以外のアプリでキーバインディングを変えるのは非常に煩わしいので、Karabiner-Elementsを使ってMacOSと同じキーバインディングでLEIMも使えるように設定する。

1.3.1 設定

MacOSは C-\ 以外のキーバインディングにしておく。
(この場合は Control + Option + Space に設定してある)

以下のJSONファイルを作成してからKarabiner-Elementsで当該ルールを有効化する。

{
  "title": "IM",
  "rules": [
    {
      "description": "Spacemacs以外でもC-\\で入力モードを切り替える",
      "manipulators": [
        {
          "type": "basic",
          "from": {
            "key_code": "backslash",
            "modifiers": {
              "mandatory": [
                "left_control"
              ]
            }
          },
          "to": [
            {
              "key_code": "spacebar",
              "modifiers": [
                "left_control",
                "left_option"
              ]
            }
          ],
          "conditions": [
            {
              "type": "frontmost_application_unless",
              "bundle_identifiers": [
                "Emacs"
              ]
            }
          ]
        }
      ]
    }
  ]
}

これでSpacemacs以外のアプリでも C-\ でIMの切り替えができるようになる。
